## 加速渲染 1.0.11-1.20.1 ALPHA
- 增强了模型渲染性能.
- 增强了文字加速性能 (受 [Modern UI](https://www.curseforge.com/minecraft/mc-mods/modern-ui) 启发, 感谢冰月).
- 添加了 Modern UI 加速支持.
- 添加了基于 `InterModComms` 的加速黑名单 API.
- 修复了 ImmediatelyFast 兼容性问题
- 修复了被过滤的物品在 GUI 中的渲染问题
- 修复了物品耐久条和冷却渲染问题
- 修复了物品层渲染顺序问题.
- 修复了 Certain Questing Additions 的物品渲染问题.

## 注意事项
- 1.20.1移植的游戏内配置界面需要安装Configured以启用.
- 1.20.1移植的GeckoLib加速需要安装MixinBooster以启用.
- 1.20.1移植的车万女仆加速需要安装MixinBooster以启用.

## Accelerated Rendering 1.0.11-1.20.1 ALPHA
- Improves mesh rendering performance.
- Improves text acceleration performance (Inspired by [Modern UI](https://www.curseforge.com/minecraft/mc-mods/modern-ui), special thanks to BloCamLimb).
- Adds acceleration support for Modern UI.
- Adds acceleration blacklist API based on `InterModComms`
- Fixes ImmediatelyFast compatibility issue.
- Fixes filtered item incorrect rendering issue in GUI.
- Fixes item durability bar and cooldown overlay rendering issue.
- Fixes item layer rendering order issue.
- Fixes item rendering issue of Certain Questing Additions.

## Important Notes
- 1.20.1 port requires Configured to modify configurations in game.
- 1.20.1 port of GeckoLib acceleration requires MixinBooster to enable.
- 1.20.1 port of TouhouLittleMaid acceleration requires MixinBooster to enable.
